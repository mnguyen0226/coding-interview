# Solution to all warm-up challenges

## Challenge 1: 
```
Write a C program to print
*
**
***
****
on screen.
### Solution: Use the for loop
```
```c++
#include <stdio.h>

int main(){
    printf("*\n");
    printf("* *\n");
    printf("* * *\n");
    printf("* * * *\n");

    return 0;
}
```
## Challenge 2:
Store two integers in two variables x and y. Print the sum of the two
```c++
#include <stdio.h>
int main(){
    int x = 1;
    int y = 2;
    printf("Sum is %d", x + y);

    return 0;
}
```
