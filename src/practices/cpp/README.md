# 1. Solutions for Print Challenges
### Challenge 1:
```
Write a C++ program to print
*
**
***
****
on screen
```
```c++
#include <iostream>
using namespace std;

int main(){
    cout << "*" << endl;
    cout << "**" << endl; 
    cout << "***" << endl; 
    cout << "****" << endl; 
    return 0;
}
```

### Challenge 2:
Store an integer in a variable x and print it on the screen.
```c++
#include <iostream>
using namespace std;

int main()
{
    int x;
    cout << "Please enter a variable: ";
    cin >> x;
    cout << "Value is: " << x;

    return 0;
}
```
